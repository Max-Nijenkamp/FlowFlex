# Projects & Work

Projects & Work is the work management domain of FlowFlex. It provides everything a team needs to plan, execute, and deliver work — from individual tasks and agile sprints to gantt charts, OKRs, project wikis, and portfolio dashboards. The projects panel lives at `/projects`.

## Navigation Groups

- **Work** — Tasks, Kanban, Sprints, Approvals
- **Planning** — Milestones, Gantt, OKRs, Portfolios
- **Resources** — Time Tracking, Documents, Wikis
- **Settings** — Statuses, Labels, Workflows

## Modules

| Module | File | Module Key | Description |
|---|---|---|---|
| Tasks | [[tasks]] | `projects.tasks` | Task creation, assignment, status, subtasks, comments, attachments |
| Sprints | [[sprints]] | `projects.sprints` | Agile sprint management, backlog, velocity, burndown |
| Kanban | [[kanban]] | `projects.kanban` | Visual kanban board — custom page with drag and WIP limits |
| Milestones | [[milestones]] | `projects.milestones` | Project milestones, due dates, completion dependencies |
| Time Tracking | [[time-tracking]] | `projects.time` | Time entries against tasks/projects, billable hours, reports |
| Documents | [[documents]] | `projects.documents` | Project document attachments, upload, version, link to tasks |
| Gantt | [[gantt]] | `projects.gantt` | Gantt chart — custom page with 90-day window and status colours |
| Portfolios | [[portfolios]] | `projects.portfolios` | Multi-project portfolio view, aggregate progress, resources |
| OKRs | [[okrs]] | `projects.okrs` | Objectives and Key Results, quarterly cycles, progress check-ins |
| Wikis | [[wikis]] | `projects.wikis` | Project-scoped wiki pages, rich text, search |
| Approvals | [[approvals]] | `projects.approvals` | Approval workflows for project deliverables |
