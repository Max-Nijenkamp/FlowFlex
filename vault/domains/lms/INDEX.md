# Learning & Development (LMS)

FlowFlex's LMS panel replaces standalone learning platforms by embedding course authoring, compliance tracking, skills management, and mentoring directly inside the same workspace employees use every day.

**Panel:** `lms` — `/lms`
**Filament Color:** Green
