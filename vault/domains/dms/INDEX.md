# Document Management (DMS)

FlowFlex's DMS panel provides a centralised document repository with collaboration, approval workflows, retention policies, and a template library — replacing SharePoint, Box, and Notion for document management use cases.

**Panel:** `dms` — `/dms`
**Filament Color:** Slate
