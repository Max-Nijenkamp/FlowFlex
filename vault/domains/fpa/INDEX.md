# Financial Planning & Analysis (FP&A)

FlowFlex's FP&A panel provides budgeting, rolling financial forecasts, budget-vs-actual variance analysis, executive KPI scorecards, and headcount planning — replacing Anaplan and Adaptive Insights for mid-market finance teams.

**Panel:** `fpa` — `/fpa`
**Filament Color:** Emerald
