# Product-Led Growth (PLG)

FlowFlex's PLG panel consolidates the tooling needed to run a product-led growth motion — free trial management, in-app onboarding, activation tracking, feature flags, usage analytics, and an in-app changelog — replacing external tools like Pendo, Appcues, and LaunchDarkly.

**Panel:** `plg` — `/plg`
**Filament Color:** Sky

| Module | Key | Nav Group | Description |

| [[onboarding-flows]] | plg.onboarding | Onboarding | In-app onboarding tours with step sequences and completion tracking |
| [[activation-metrics]] | plg.activation | Analytics | Activation funnel: define activation events, track cohort rates |
| [[feature-flags]] | plg.flags | Analytics | Feature flag management with company/user targeting and gradual rollouts |
