# Whistleblowing & Ethics

FlowFlex's Ethics panel provides a confidential, EU Whistleblowing Directive-compliant incident reporting system — anonymous report intake, investigation case management, policy acknowledgments, and program analytics — replacing NAVEX and EthicsPoint.

**Panel:** `ethics` — `/ethics`
**Filament Color:** Red
