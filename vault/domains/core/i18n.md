# Internationalisation

> Locale detection, `SetLocale` middleware, and translation management â€” the entire FlowFlex UI adapts to each user's language and the company's default locale.

**Panel:** `app`
**Module key:** `core.i18n`

## What It Does

The Internationalisation module handles locale resolution for every request in the app panel. Each user can set their personal locale preference; if they have not, the company's default locale from Company Settings is used. The `SetLocale` middleware reads the resolved locale and calls `App::setLocale()` before the request reaches any controller or Filament component. All translatable strings in FlowFlex use Laravel's `__()` helper against locale files stored in `lang/{locale}/`. The module also provides a translation management UI in the admin panel for maintaining and adding locale files.

## Features

### Core
- Supported locales: `en` (English), `nl` (Dutch), `de` (German), `fr` (French), `es` (Spanish)
- Locale resolution order: (1) user personal preference, (2) company default locale, (3) `en` fallback
- `SetLocale` middleware: runs on all app panel routes, calls `App::setLocale()` and `Carbon::setLocale()`
- User locale preference stored on `users.locale` column â€” selectable from account settings page
- Company default locale stored on `companies.locale` â€” set in Company Settings

### Advanced
- Number formatting respects locale: decimal separator, thousands separator, currency symbol position
- Date and time formatting respects locale: `d MMMM YYYY` for `nl`/`de`, `MMM d, YYYY` for `en`
- RTL support scaffolded for future Arabic/Hebrew locales â€” Filament panel direction configurable per locale
- Translation fallback: missing keys in the requested locale fall back to `en` rather than showing raw key names
- Translation file hot-reload in local environment: `php artisan lang:sync` regenerates compiled locale files

### AI-Powered
- AI-assisted translation: new UI strings added to `en` locale files automatically queued for AI translation to all supported locales â€” flagged for human review before publishing
- Translation coverage dashboard: admin panel shows percentage of strings translated per locale
