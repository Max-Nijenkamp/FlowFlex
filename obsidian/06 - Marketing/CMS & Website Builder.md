# CMS & Website Builder

Block-based CMS for marketing websites. Includes blog, SEO, media library, and headless API.

**Who uses it:** Marketing team, content editors
**Filament Panel:** `marketing`
**Depends on:** Core
**Phase:** 4
**Build complexity:** Very High — 3 resources, 2 pages, 6 tables

## Features

- **Block-based page builder** — visual drag-and-drop sections
- **Blog / news** — article management with categories, tags, author bio
- **SEO meta fields** — title, description, canonical URL, OG image per page
- **Media library** — centralised image and file management
- **Multi-language (i18n)** — content in multiple languages
- **Scheduled publishing** — set a future publish date
- **Redirect manager** — manage 301/302 redirects
- **Template library** — reusable page templates
- **A/B test pages** — split traffic between two page variants
- **Headless CMS API** — REST API for decoupled frontends (React, Next.js, etc.)
