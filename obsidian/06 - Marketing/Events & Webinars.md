# Events & Webinars

Build registration pages, manage attendees, and automate event communications.

**Who uses it:** Marketing team, event managers
**Filament Panel:** `marketing`
**Depends on:** Core
**Phase:** 5

## Features

- **Event registration page builder**
- **Attendee management** — registration list, check-in status
- **Reminder email/SMS sequences** — automated before the event
- **QR code check-in** — mobile-based on-site check-in
- **Post-event follow-up automation** — email sequence to attendees
- **Webinar platform integration** — Zoom, Teams
