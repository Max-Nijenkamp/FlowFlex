# Forms & Lead Capture

Build and embed forms anywhere. Every submission can auto-create a CRM contact.

**Who uses it:** Marketing team
**Filament Panel:** `marketing`
**Depends on:** [[Contact & Company Management]] (for auto-create)
**Phase:** 4

## Features

- **Drag-and-drop form builder**
- **Field types** — text, email, phone, date, dropdown, checkbox, file upload, signature
- **Conditional logic** — show/hide fields based on responses
- **Embed** via iframe or JavaScript snippet
- **On submit:** auto-create CRM contact, trigger workflow, fire notification
- **Spam protection** — CAPTCHA integration
- **Multi-step forms** — split long forms across multiple pages
