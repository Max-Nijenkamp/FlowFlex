# Mentoring & Coaching

Structured mentoring programme. Match mentors and mentees, track sessions, run cohorts.

**Who uses it:** L&D team, employees
**Filament Panel:** `lms`
**Depends on:** [[Employee Profiles]]
**Phase:** 5

## Features

- **Mentor matching** — skills and goals-based pairing algorithm
- **Session scheduling** — booking via [[Booking & Appointment Scheduling]]
- **Progress tracking** — session notes and goal progress
- **Programme management** — cohort-based programmes
- **Internal coaching network directory** — browsable coach profiles
