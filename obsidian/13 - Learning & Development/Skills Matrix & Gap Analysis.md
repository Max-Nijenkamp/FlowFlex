# Skills Matrix & Gap Analysis

Know what skills your team has, what each role needs, and bridge the gap with training.

**Who uses it:** L&D team, HR, managers
**Filament Panel:** `lms`
**Depends on:** [[Employee Profiles]], [[Course Builder & LMS]]
**Phase:** 5

## Features

- **Skills library (taxonomy)** — structured list of all skills in the organisation
- **Role skill requirements** — define what skills each role needs
- **Employee skill self-assessment** — employees rate their own proficiency
- **Gap analysis dashboard** — where are the shortfalls?
- **Training recommendations** based on gaps (linked to course catalogue)
