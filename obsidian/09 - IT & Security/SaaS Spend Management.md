# SaaS Spend Management

Discover what tools the company is actually using and stop paying for things nobody uses.

**Who uses it:** IT team, finance
**Filament Panel:** `it`
**Phase:** 5

## Events Fired

- `SaaSLicenceExpiring` → consumed by [[Finance Overview]] (alerts finance team), Notifications (creates renewal task)

## Features

- **SaaS discovery** — detects all tools in use, including shadow IT (via SSO / expense analysis)
- **Spend tracking** — monthly cost per tool
- **Licence optimisation recommendations** — "15 users have access but only 3 logged in last month"
- **Renewal calendar** — upcoming renewals across all tools
- **Usage analytics** per tool
- **Alerts for duplicate tools** or unused licences
