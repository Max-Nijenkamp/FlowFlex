# Internal IT Helpdesk

Employee-facing portal for IT issues. Separate from customer support — this is for internal staff.

**Who uses it:** All employees (submit), IT team (resolve)
**Filament Panel:** `it`
**Phase:** 5

## Features

- **Employee-facing portal** for IT issue submission
- **Hardware fault reporting** — broken laptop, printer, etc.
- **Software request** — need a new tool or access
- **Access request management** — request permissions to a system
- **SLA tracking** — response and resolution targets
- **Knowledge base** for self-service resolution (links to [[Knowledge Base & Wiki]])
