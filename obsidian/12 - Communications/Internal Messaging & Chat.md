# Internal Messaging & Chat

Real-time channels and direct messages. The Slack alternative for teams that want everything in one place.

**Who uses it:** All employees
**Filament Panel:** `communications`
**Phase:** 5

## Features

- **Real-time channels** — public and private topic channels
- **Direct messages** — one-to-one and group DMs
- **Threads** — reply in thread to keep channels clean
- **File sharing**
- **Task linking** — reference a task from a message
- **Search history** — full message search
- **Reactions** — emoji reactions to messages
