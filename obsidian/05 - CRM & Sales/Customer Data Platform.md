# Customer Data Platform (CDP)

Unified customer profile across all modules. The single source of truth for every customer interaction.

**Who uses it:** Sales, marketing, support, analytics
**Filament Panel:** `crm`
**Depends on:** [[Contact & Company Management]]
**Phase:** 5

## Features

- **Unified customer profile** — merges data from CRM, Finance, E-commerce, Support
- **Dynamic segmentation** — create segments based on any combination of attributes or behaviours
- **Behaviour event tracking** — website visits, product usage, email interactions
- **Source attribution** — where did this customer come from?
- **Identity resolution** — recognises the same person across different modules
- **Syncs to [[Email Marketing]] module** for targeted campaigns
