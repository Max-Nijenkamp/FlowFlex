# Team Velocity & Operations Metrics

Operational performance metrics. Cycle times, throughput, and early burnout signals.

**Who uses it:** Engineering managers, operations managers, HR
**Filament Panel:** `analytics`
**Phase:** 5

## Features

- **Cycle time tracking** — time from task start to completion
- **Throughput charts** — tasks/stories completed per week
- **Bottleneck detection** — where work is piling up
- **Predictive delivery estimates** — when will this project complete at current velocity?
- **Cross-team comparison** — performance across teams
- **Burnout signal flags** — correlate with [[Employee Feedback]] data
