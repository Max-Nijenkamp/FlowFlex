# Custom Dashboards

Drag-and-drop dashboard builder. Every team sees the metrics relevant to them.

**Who uses it:** All roles (personalised dashboards), managers, leadership
**Filament Panel:** `analytics`
**Phase:** 5

## Features

- **Drag-and-drop widget builder** — add, resize, reorder widgets
- **Widget types** — metric card, line chart, bar chart, donut chart, table, funnel
- **Data sources** — from any active module (HR metrics, Finance metrics, CRM metrics, etc.)
- **Role-based dashboard views** — each team sees their relevant metrics
- **Auto-refresh intervals** — configurable per dashboard
- **Share or embed dashboard** — external link for stakeholders
