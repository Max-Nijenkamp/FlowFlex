# Report Builder

Self-serve reporting for every team. No SQL knowledge required.

**Who uses it:** All roles
**Filament Panel:** `analytics`
**Phase:** 5

## Features

- **Self-serve report builder** — no SQL needed
- **Operations** — filter, group, sort, pivot on any data
- **Cross-module data joins** — combine HR + Finance data in one report
- **Scheduled report delivery** — auto-email as PDF or Excel on cadence
- **Saved report library** — save and share reports with the team
