# Data Warehouse & Export

Enterprise data export to external BI tools. For companies that need raw data access.

**Who uses it:** Data engineers, analysts (Enterprise tier)
**Filament Panel:** `analytics`
**Phase:** 6 (Enterprise & Scale)

## Features

- **BigQuery sync** — scheduled export to Google BigQuery
- **Snowflake connector** — bidirectional data sync
- **S3 / Azure Blob export** — raw data file export
- **Scheduled ETL jobs** — configurable export cadence
- **Data retention policy management** — control what data is exported and for how long
